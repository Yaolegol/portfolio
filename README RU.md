# Старт проекта

## 1. Install dependencies
> npm i

## 2. Build project
> npm run build

## 3. Start project
> cd ./server

> npm i

> npm start

Project start on **8000** port

## 4. Наслаждайтесь!

# О проекте

### Описание
Landing page

- выполнена с использованием react, less, swiper
- адаптивная верстка
