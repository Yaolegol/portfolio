# Старт проекта

##1. Установить зависимости
> npm i

##2. Запустить проект
> npm start

Project start on **8080** port

##3. Наслаждайтесь!

### Описание

- адаптивная верстка
- возможность выбора локали (en/ru)

### Серверная часть
